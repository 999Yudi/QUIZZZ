CRIMES CIBERNETICOS

Qual das alternativas abaixo melhor descreve o que é um ataque de DDoS?

  • Um ataque que tenta derrubar um site ou serviço com excesso de acessos simultâneos.
  • Um tipo de phishing que usa redes sociais.
  • Um malware que criptografa arquivos para pedir resgate.
  • Um vírus que rouba senhas.

O que significa o termo malware?

  • Programa utilizado para melhorar a segurança dos sistemas.
  • Malware é um tipo de hardware que ajuda a proteger o computador contra ataques virtuais.
  • Malware só pode infectar computadores com sistema operacional Windows, pois outros sistemas como Linux e macOS são completamente imunes.
  • Software malicioso criado para causar danos ou roubar informações.
